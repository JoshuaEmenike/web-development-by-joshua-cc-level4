HTML Tags and CSS Properties

This page demonstrates some basic HTML tags and CSS properties.

HTML Tags

The following are examples of HTML tags:

<h1> to <h6>

The <h1> to <h6> tags are used for creating headings. <h1> is the highest level and <h6> is the lowest.

<p>

The <p> tag is used to define a paragraph of text.

<a>

The <a> tag defines a hyperlink.

<ul> and <li>

The <ul> tag defines an unordered list, and <li> defines each list item.

<div>

The <div> tag is used to group content together.

<img>

The <img> tag is used to display images on a webpage.

<footer>

The <footer> tag defines the footer section of a page.

<header>

The <header> tag is used to define introductory content or a navigation menu.

<section>

The <section> tag is used for grouping content related to a specific topic.

<span>

The <span> tag is used to group inline elements.

CSS Properties

The following are basic CSS properties used to style the HTML tags:

font-family

The font-family property sets the font for the text in an element.

color

The color property defines the color of the text.

background-color

The background-color property sets the background color of an element.

margin

The margin property defines the space outside an element.

padding

The padding property defines the space inside an element.

border

The border property defines the border around an element.

font-size

The font-size property sets the size of the font.

text-align

The text-align property defines the alignment of text within an element.
